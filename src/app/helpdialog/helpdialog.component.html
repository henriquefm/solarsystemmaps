About

This project tries to give an idea of the scale of the Solar System by showing how big it would be, in Google Maps, if the Earth were the size of a basketball or other objects.

It was inspired by this video by Mark Rober: https://www.youtube.com/watch?v=pR5VJo5ifdE

Notes:

  • Click anywhere on the map to change the Sun's position
  • The planets' orbits shown are ellipses, with each planet's real eccentricity

Github page - https://github.com/henriquefm/solarsystemmaps
